Class Objective

This course is for students familiar with HTML & CSS who want to learn the latest generation of browser-based technologies associated with HTML5. Topics covered include new structural and semantic elements of HTML5, native containers for audio and video, styling and animations with CSS3, accessibility considerations, differences in browser implementation, and an introduction to the power of JavaScript API's.

Learning Outcomes

At the conclusion of this course, the student should be able to:

  • Use new HTML5 elements to effectively structure a webpage's contents for optimal accessibility and semantic meaning.
  • Utilize new CSS selectors for improved targeting control with minimal markup.
  • Style pages using CSS techniques without relying on presentational images.
  • Employ progressive enhancement techniques for optimal user experience.
  • Understand how to use of boilerplates & frameworks to start new projects.
  • Be familiar with the related technologies associated with HTML5.

  • Read Chapters 5 & 8
  • This week's project is worth 2 points folks. Make sure to turn it in to pass this class!
  • I want you to flex your coding muscles and take the class sample site to a new level.
  • You can start where we left off, with the @font-face and multi-column layout already enabled.

  • Your HTML pages should make use of the following elements:
    <header> <nav> <article> <section> <aside> <footer>
  • Add at least one <audio> or <video> element. You can use the sample files included here.
  • Your CSS should make use of rgba or hsla in at least one place.
  • Recreate the navbar buttons using CSS Gradients and border-radius.
  • Create at least one media query to change something at a different screen size.
  • Create at least one hover state with a transition. This can be on the navbar links, or on any other element.
